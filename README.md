# General Expense Dashboard

A Python-based expense analytics dashboard built with Streamlit.

## Features
- Add and track expenses
- Filter by category and date
- Delete individual expenses
- View category summary
- Bar chart, pie chart, and trend chart
- Export filtered data to CSV and Excel

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- OpenPyXL
